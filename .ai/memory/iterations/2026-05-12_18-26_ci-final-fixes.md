# [ПАМЯТЬ] Сессия: Окончательное исправление ошибок CI Pipeline

2026-05-12 18:26
Финальная сессия исправления критических ошибок CI pipeline после неудачных попыток запуска тестов.

## Цель сессии
1. Исправить ошибку "unrecognized arguments: --shard=X/Y" в pytest
2. Решить проблему E2E тестов с localhost:3000 (ERR_CONNECTION_REFUSED)
3. Настроить CI для работы с публичными API и demo сайтами
4. Разделить E2E и API тесты для стабильной работы pipeline

## Выполненные действия

### 1. Исправление проблемы pytest sharding
- **Проблема**: Pytest выдавал ошибку "unrecognized arguments: --shard=X/Y"
- **Причина**: Аргумент --shard не поддерживается стандартным pytest без специальных плагинов
- **Решение**: Убрана matrix strategy с sharding, упрощён CI до простого запуска API тестов

### 2. Решение проблемы ERR_CONNECTION_REFUSED
- **Проблема**: E2E тесты пытались подключиться к http://localhost:3000/, который недоступен в CI
- **Решение**: 
  - Изменён базовый URL в settings.py на https://demo.playwright.dev/todomvc
  - Обновлён .env.example с публичными URL
  - E2E тесты выделены в отдельный workflow для ручного запуска

### 3. Реструктуризация CI workflows
- **Основной CI** (.github/workflows/e2e.yml переименован в API Tests & Linting):
  - Lint job: ruff check, ruff format, mypy
  - Test job: только API тесты с меткой @pytest.mark.api
  - Убрана matrix strategy и sharding
- **Отдельный E2E CI** (.github/workflows/e2e-tests.yml):
  - Ручной запуск и по расписанию
  - Использует публичный demo сайт playwright
  - Изолированный от основного pipeline

### 4. Добавление меток к тестам
- **Проблема**: API тесты не были помечены для селективного запуска
- **Решение**: Добавлены метки @pytest.mark.api к тестам в tests/integration/test_integration.py
  - test_get_posts_from_api
  - test_get_single_post  
  - test_create_post_via_api
  - test_integration_api_and_logging

### 5. Обновление конфигурации по умолчанию
- Изменён default base_url с localhost:3000 на https://demo.playwright.dev/todomvc
- Обновлён .env.example для использования публичных сервисов:
  - BASE_URL=https://demo.playwright.dev/todomvc
  - API_BASE_URL=https://jsonplaceholder.typicode.com
